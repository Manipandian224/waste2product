# **App Name**: Waste2Product

## Core Features:

- Animated Landing Page: An interactive and animated hero section featuring the circular economy flow, eco particle background, dynamic stats counters (waste collected, credits issued, CO₂ saved), and calls-to-action for uploading waste and exploring the marketplace.
- User Dashboard: A personalized dashboard displaying the user's Green Wallet Balance (animated counter), total waste submitted, carbon saved (circular progress), monthly waste graph, waste category pie chart, leaderboard rank badge, recent activity timeline, and an AI-calculated Eco Score Meter. All cards feature smooth fade-in animations.
- AI Waste Analyzer & Credit Calculator: An AI-powered tool that allows users to upload images of waste via drag-and-drop. The AI simulates classification to detect waste type, provide a quality score, contamination percentage, and dynamically calculates credits based on a 'Rate × Weight × Quality Score' formula, with live animation.
- Waste Pickup Scheduling: A dedicated page for scheduling waste pickups, featuring a calendar picker, time slot selector, a mock map preview, a step-by-step pickup status tracker, and an animated confirmation modal.
- Green Wallet Management: A comprehensive page to view total green credits, a detailed credit history table, carbon savings charts, and options to redeem credits for products or combine with cash, all presented with animated transaction history scrolling.
- E-commerce Marketplace: A grid-based marketplace displaying recycled products with details like image, 'Made from recycled waste' tag, waste type source, price, and credits required. Includes buy options (credits only, credits + cash, cash only), hover scale animation on product cards, and an animated cart sidebar with slide-in effects.
- AI-driven Eco Insights: An integrated module within the dashboard that uses AI as a tool to simulate and present waste trend prediction graphs, highlight most recycled categories, analyze user behavior, provide waste growth forecasts, and carbon reduction projections through animated insight cards.

## Style Guidelines:

- The primary color is an vibrant eco-green, '#00C896', reflecting sustainability and growth.
- The background color is a very dark, desaturated teal, '#192723', providing a suitable dark theme canvas for glassmorphism.
- The accent color is a lighter, slightly yellowish-green, '#66C28F', used to highlight interactive elements and provide contrast.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, techy feel, while body text will use 'Inter' (sans-serif) for objective readability, aligning with the eco-tech startup aesthetic.
- Leverage 'Lucide React' icons for a consistent, modern, and outline-based visual style that complements the clean eco-tech UI.
- Emphasize a 'modern glassmorphism' design on a dark background, ensuring all components are fully responsive across mobile and desktop devices. Content organization will follow clear grid layouts, particularly for the E-commerce Marketplace.
- Utilize 'Framer Motion' extensively for smooth, professional animations including circular flow visualizations, background particle effects, animated counters, smooth page transitions, button hover scale effects, card lift animations, fade-in effects for UI elements, loading skeleton animations, slide animations for cart elements, and animated insights and charts for a dynamic user experience.
